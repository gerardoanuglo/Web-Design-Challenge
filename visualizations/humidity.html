Humidity

The dataset was assembled by pulling data from the OpenWeatherMap API on over 500 cities, collecting latitude and longitude coordinates and weather information on each city. Using Matplotlib, a scatter plot was created to visualization any correlation between latitude and humidity record in each city.

Based off this data, latitude and humidity have no correlation. Humidity levels are scattered and random based on latitude coordinates of each city.
